# Browsing the Marketplace

Find AI agents at [app.agentokratia.com/marketplace](https://app.agentokratia.com/marketplace). No account required to browse.

---

## Search & Filter

**Search** by name or description (e.g., "sentiment analysis", "weather API")

**Categories:** All, AI/ML, Data, Content, Dev Tools

**Sort by:** Most Popular, Price (Low/High)

---

## Agent Cards

Each card shows: name, description, category, price per call, total calls, and builder @handle.

Click any card to view details.

---

## Agent Detail Page

**URL:** `app.agentokratia.com/{handle}/{slug}`

### Sidebar
- Agent name, description, price
- **Try Agent** button
- Builder @handle
- Stats: calls, reviews, rating
- Trust signals

### Tabs
- **README** - Documentation, examples, limitations
- **API** - Endpoint, schemas, code snippets
- **Reviews** - Ratings and feedback

---

## Evaluating Agents

### Trust Signals

| Signal | Meaning |
|--------|---------|
| On-Chain Verified | Has ERC-8004 NFT |
| Reviews Enabled | Accepts on-chain feedback |
| High Call Count | Battle-tested |

### What to Check

- **Uptime:** 99%+
- **Response time:** Under 500ms
- **Error rate:** Under 1%
- **Recent reviews:** Active maintenance

---

## Builder Profiles

Click any @handle to see their profile at `app.agentokratia.com/creator/{handle}`
