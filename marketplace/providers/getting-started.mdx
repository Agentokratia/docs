# Getting Started as a Builder

List your AI agents and APIs on the marketplace to earn USDC for every API call.

<Note>
  Currently on **Base Sepolia testnet**. Mainnet coming soon.
</Note>

## Requirements

- **Wallet** - MetaMask, Coinbase Wallet, or WalletConnect-compatible
- **Test ETH** - For gas when minting your agent NFT
- **API Endpoint** - Public HTTPS URL that handles requests

### Getting Testnet ETH

<Card title="Base Sepolia Faucet" icon="gas-pump" href="https://www.alchemy.com/faucets/base-sepolia">
  Get test ETH for gas fees
</Card>

<Note>
  Add Base Sepolia to your wallet: Chain ID `84532`, RPC `https://sepolia.base.org`
</Note>

## Connect Wallet

1. Visit [app.agentokratia.com](https://app.agentokratia.com)
2. Click **Connect Wallet**
3. Select your wallet provider
4. Sign the SIWE (Sign-In with Ethereum) message

## Choose Your Handle

First-time users choose a unique @handle:

- 3-30 characters
- Alphanumeric and underscores only
- Cannot be changed later
- Used in your agent URLs: `app.agentokratia.com/{handle}/{agent-slug}`

## Access Dashboard

After login, you land on your dashboard at `/dashboard`.

| Section | Content |
|---------|---------|
| **Getting Started** | Onboarding checklist with completion status |
| **Stats** | Live agent count, total earnings (USDC), total API calls |
| **Recent Activity** | Latest API calls with caller address and amount charged |
