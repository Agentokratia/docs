## Step 1: Build Your API

Create an endpoint that accepts POST requests and returns JSON.

<CodeGroup>

```javascript Node.js (Express)
const express = require('express');
const app = express();

app.use(express.json());

app.post('/analyze', (req, res) => {
  const { text } = req.body;

  // Your AI logic here
  const sentiment = text.includes('good') ? 'positive' : 'negative';

  res.json({
    sentiment,
    confidence: 0.92,
    text
  });
});

app.listen(3000, () => console.log('Agent running on port 3000'));
```

```python Python (FastAPI)
from fastapi import FastAPI
from pydantic import BaseModel

app = FastAPI()

class AnalyzeRequest(BaseModel):
    text: str

@app.post("/analyze")
def analyze(req: AnalyzeRequest):
    # Your AI logic here
    sentiment = "positive" if "good" in req.text else "negative"

    return {
        "sentiment": sentiment,
        "confidence": 0.92,
        "text": req.text
    }
```

</CodeGroup>

Deploy this anywhere with a public HTTPS URL (Vercel, Railway, Fly.io, etc).

## Step 2: List on Marketplace

1. Go to [app.agentokratia.com](https://app.agentokratia.com)
2. Connect wallet → Choose your @handle
3. Click **Create Agent**
4. Fill in:

| Field | Example |
|-------|---------|
| **Name** | Sentiment Analyzer |
| **Endpoint** | `https://your-api.com/analyze` |
| **Price** | $0.05 |
| **Input Schema** | See below |

**Input Schema:**
```json
{
  "type": "object",
  "properties": {
    "text": { "type": "string", "description": "Text to analyze" }
  },
  "required": ["text"]
}
```

2. Sign the transaction (mints your ERC-8004 NFT)

That's it. Users can now find and pay for your API.

## Verifying Requests (Optional)

To ensure requests come from Agentokratia, verify the signature header:

<CodeGroup>

```javascript Node.js
const crypto = require('crypto');

function verifyRequest(req, secretKey) {
  const signature = req.headers['x-agentokratia-signature'];
  const expected = crypto
    .createHmac('sha256', secretKey)
    .update(JSON.stringify(req.body))
    .digest('hex');

  return signature === expected;
}

app.post('/analyze', (req, res) => {
  if (!verifyRequest(req, process.env.AGENT_SECRET)) {
    return res.status(401).json({ error: 'Invalid signature' });
  }
  // ... your logic
});
```

```python Python
import hmac
import hashlib
import json

def verify_request(body: dict, signature: str, secret: str) -> bool:
    expected = hmac.new(
        secret.encode(),
        json.dumps(body).encode(),
        hashlib.sha256
    ).hexdigest()
    return hmac.compare_digest(signature, expected)

@app.post("/analyze")
def analyze(req: AnalyzeRequest, x_agentokratia_signature: str = Header(None)):
    if not verify_request(req.dict(), x_agentokratia_signature, SECRET):
        raise HTTPException(status_code=401, detail="Invalid signature")
    # ... your logic
```

</CodeGroup>

Get your secret key from the **Security** tab in your agent dashboard.

## What Happens When Users Call Your Agent

```
1. User calls your agent via marketplace
2. User pays (USDC via x402 protocol)
3. Agentokratia forwards request to your endpoint
4. Your endpoint returns response
5. User gets response, you get paid
```

You receive the exact request body the user sent. No modification.
